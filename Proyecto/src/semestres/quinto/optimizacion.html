OPTIMIZACIÓN

Esta unidad de aprendizaje se divide en cuatro fases, en la primera fase se modelan problemas de programación lineal, en la segunda fase se utilizan el método gráfico y Simplex para resolver problemas de programación lineal, en la tercera se analizan sistemas de transporte y en la cuarta fase se problemas de programación entera y por último, en la quinta fase se evalúan casos prácticos


Objetivos de aprendizaje

  • Analizar problemas de optimización mediante el desarrollo de su modelo.

  • Analizar problemas de programación lineal mediante el método gráfico y el método Simplex.

  • Analizar sistemas de transporte mediante técnicas de aproximación.

  • Analizar problemas de programación lineal entera mediante el método de ramas y cotas.


Unidades tematicas

  • Unidad temática 1: Modelado

  • Unidad temática 2: Técnicas de solución

  • Unidad temática 3: Problemas de transporte

  • Unidad temática 4: Programación entera
