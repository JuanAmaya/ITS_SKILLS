TOPICOS SELECTOS DE PROGRAMACIÓN

Optativa III FBP

La unidad de aprendizaje de Tópicos Selectos de Programación está diseñado para orientar a los estudiantes en el entendimiento de los principales elementos teórico prácticos de las tecnologías para el desarrollo de aplicaciones web, manejará diversas herramientas para el desarrollo de aplicaciones web como: Diseño de páginas, Protocolos TCP/IP, Lenguaje XML y Uso De Java Script.


Objetivos de aprendizaje

  • Determinar el uso de algunos lenguajes de programación.

  • Analizar los diferentes protocolos mediante herramientas de comunicación.

  • Realizar proyectos individuales y por equipo.

  • Utilizar las herramientas necesarias mediante su aplicación eficiente.

  • Aplicar XML mediante el acceso a una base de datos para guardar y obtener información.


Unidades tematicas

  • Unidad temática 1: Protocolo TCP/IP y herramientas de comunicación en Internet

  • Unidad temática 2: Principales herramientas para el desarrollo de páginas en Internet

  • Unidad temática 3: Java Script: principales instrucciones

  • Unidad temática 4: Lenguaje de Etiquetado Extensible: XML
