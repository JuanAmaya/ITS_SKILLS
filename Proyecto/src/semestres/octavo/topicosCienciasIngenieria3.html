TÓPICOS SELECTOS DE CIENCIAS DE LA INGENIERÍA III

Optativa VII FP

En esta unidad de aprendizaje el estudiante será capaz de identificar patrones de información mediante técnicas de minería de datos, en la primer fase conocerá como se trabaja con logs de información y construirá redes de información para los procesos analizados, en la segunda fase aplicara técnicas de minería de datos que le permitan identificar patrones de información y en la tercera fase aprenderá técnicas para el procesamiento y manejo de datos.


Objetivos de aprendizaje

  • Identificar los elementos utilizados para construir diagramas de proceso.

  • Analizar la aplicación de técnicas de minería de datos mediante el uso de herramientas para minería de procesos.

  • Analizar la estructura de los datos de proceso mediante técnicas establecidas.


Unidades tematicas

  • Unidad temática 1: Modelado y Análisis de Procesos

  • Unidad temática 2: Técnicas de Minería de Datos

  • Unidad temática 3: Manejo y procesamiento de datos
